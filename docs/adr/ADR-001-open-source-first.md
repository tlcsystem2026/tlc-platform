# ADR-001 Open Source / Free Resource First

Status: Accepted

## Decision
采用“优先复用，必要开发；开源优先，免费优先；成熟优先，简单优先；安全完整，不追求大而全”。

## Selection gate
1. 现有组件能否满足？
2. 成熟开源能否直接采用？
3. 免费资源/知识库/Tools 能否安全满足？
4. 轻量集成是否优于自研？
5. 仅公司特有业务规则才优先自研。

## Constraints
- 不因“开源”而无限增加组件。
- 新组件必须减少总开发/运维成本。
- 必须可备份、可升级、可替换、可审计。
- 真实业务数据不得为换取“免费”而进入未经批准的外部服务。
