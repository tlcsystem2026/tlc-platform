# Build034 Business Requirements Baseline

## P0-1 销售请求书比较
- 原始Excel与PDF自动比较。
- 错误时生成错误文件，并可在Web查看。
- 无错误时进入待审核数据库。

## P0-2 审核与正式销售台账
- 审核是否属实、是否重复、是否取消。
- 重复或取消可在页面指定取消并保留审计信息。
- 审核通过后进入正式销售请求书/销售一览台账。
- 必须幂等，重复执行不得重复登记。

## P0-3 银行流水导入
- 支持多家银行。
- 导入优先顺序：CSV、Excel、PDF。
- 每次导入形成可追溯批次。
- 银行格式由业务方提供后建立Adapter。

## P0-4 客户对账
- 按客户请求截止日期过滤正式请求数据。
- 按银行入金截止日期过滤银行入金数据。
- 判断结清、部分入金、未付账款。
- 结清后可指定下一期两个截止日期。

## P1 采购
- 采购请款书支持CSV/Excel/PDF，图像先定义接口。
- 后续实现采购请款书与银行流水核对。
